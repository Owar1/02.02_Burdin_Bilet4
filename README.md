Описание проекта
WPF-приложение для расчета стоимости жалюзи с возможностью генерации текстовых чеков. Приложение разработано в рамках экзаменационного задания.

Функциональность
1. Расчет стоимости
Ввод размеров (ширина и высота) в метрах
Выбор материала: Алюминий (15.50 руб/м²) или Пластик (9.90 руб/м²)
Автоматический расчет стоимости по формуле: стоимость = ширина × высота × цена_материала

Валидация ввода:
Проверка на пустые поля
Проверка корректности числового формата
Проверка на положительные значения

2. Генерация квитанции
Создание текстового чека в формате ООО "Уютный Дом"
Автоматическое сохранение на Рабочий стол
Формат имени файла: УникальныйНомерЧека_Дата_Стоимость.txt
Чек содержит:
Реквизиты компании
Номер чека и дату
Детали заказа (размер, материал, площадь, стоимость)
Итоговую сумму

Требования к системе
.NET 6.0 или выше
Windows 10/11
Visual Studio 2022 (для сборки)
